Using Single board computers, single board microcontrollers and invention kits technologies
Raspberry Pi, Arduinos, Makey Makeys and other small computing devices are reshaping the IT
world and this is the new era of electrical and computer engineering.
Presently, Arduino, Raspberry pi and Makey Makeys are most popular electronic
hardware kits among the projects and the user community. Those silent, energy efficient, simple and
low cost hard ware kits are giving opportunity to any non-engineer person to use them on their
creating digital projects. Let’s discussed about those three giants general information and technical
specifications.

1. Arduino – Arduino is an open source hard ware and software company. They designs and
manufactures single board microcontrollers and microcontroller kits for building digital devices
projects and user community. Arduino board design use a variety of microprocessors and
controllers. The microcontrollers can be programmed using the C and C++ programming languages.
Using a standard API which is also known as the “Arduino Language”.

2. Raspberry pie-Raspberry pie is a series of small single board(SBCs)developed by in the UK by
Raspberry Pi Foundation. This Project Originally leaned towards the promotion of teaching basic
computer science in school and in developing countries. It provides Raspbian OS and as well as third
parties OS. Also it promotes python and scratch as main languages.3. Makey Makeys- Makey Makey-An Invention Kit for Everyone.This is an invention kit designed
to connect everyday objects to computer keys. Using a circuit board, alligator clips, and a USB cable,
the toy uses closed loop electrical signals to send the computer either a keyboard stroke or mouse
click signal. This function allows the Makey Makey to work with any computer program or webpage
that accepts keyboard or mouse click inputs and then it will work.
